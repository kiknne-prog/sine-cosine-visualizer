正弦余弦曲线可视化

初中数学知识点 - 三角函数图像与性质

sin(x)
cos(x)
坐标: (0, 0)
角度: 0° (0 弧度)

参数控制

y = 1 × sin(1x + 0) + 0
y = 1 × cos(1x + 0) + 0

常用函数预设

函数性质

  • 定义域：R (所有实数)
  • 值域：[-A+k, A+k]
  • 周期：T = 2π/ω
  • 对称性：sin(x)关于原点对称，cos(x)关于y轴对称

特殊角度值

角度 弧度 sin值 cos值
0° 0 0 1
30° π/6 1/2 √3/2
45° π/4 √2/2 √2/2
60° π/3 √3/2 1/2
90° π/2 1 0

图像特点

  • 振幅 (A)：控制曲线的最大偏离程度
  • 频率 (ω)：控制曲线的疏密程度
  • 相位 (φ)：控制曲线的左右平移
  • 偏移 (k)：控制曲线的上下平移
  • 关系：cos(x) = sin(x + π/2)
